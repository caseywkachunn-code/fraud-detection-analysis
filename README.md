📊 Fraud Detection & Transaction Pattern Analysis
📌 Overview

This project analyses transaction data to identify fraud patterns using SQL and Python. It focuses on detecting high-risk categories and time-based fraud behaviour.

⚙️ Key Features
Data cleaning and preprocessing
SQL-based fraud analysis
Time-based feature engineering
Fraud pattern detection
Data visualisation

📈 Key Insights
Fraud is concentrated in high-frequency categories such as grocery and online shopping
Peak fraud activity occurs between 22:00 and 03:00
Daytime fraud levels are significantly lower

🛠️ Tech Stack
Python (pandas, matplotlib)
SQL (SQLite)
